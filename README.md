# MarioAdventureGame(2020)
made with unity 5.0

### HOW TO DOWNLOAD GAME ###
> 1. Download <b>"MarioRun.html"</b>, <b>"MarioRun.unity3d"</b> files
> 2. Open the <b>"MarioRun.html"</b> file using Internet Explorer
> 3. Install <b>Unity Web Player</b> if necessary
